# Uber Data Analysis

A data analysis project focused on exploring Uber ride data to identify travel patterns, peak booking hours, and customer behavior trends.

## What it does

* Cleans and preprocesses Uber trip data
* Extracts date and time-based features from ride records
* Analyzes trip frequency across hours, days, and categories
* Visualizes ride distribution and travel patterns using charts and plots
* Applies feature engineering and encoding techniques for analysis

## Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## Conclusion

* Analysis revealed peak ride timings and common travel purposes
* Data visualization helped understand customer travel behavior and trip distribution
* Feature engineering improved insights into ride activity patterns
